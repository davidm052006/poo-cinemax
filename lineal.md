# PROGRAMACION LINEAL VS PROGRAMACION POO

## PRIGRAMACION LINEAL
- Es una tecnica matematica para encontrar la mejor solucion posible (optima) a un problema donde hay recursos limitados.
- Se usa mucho para maximizar ganancias o minimizar costos.
## CONCLUSION
- La programacion lineal busca la mejor combinacion posible usando matematica y logica, si necesidad de probar todas las opciones manualmente.

# PROGRMACION ORIENTADA A OBJETOS 
- Un paradigma de programacion como el orientado a objetos organiza el codigo en calses y objetos, como si fueran "cosas reales" del mundo. 
- Cada objeto tiene propiedades(datos) y acciones(metodos)
## CONLUSION
- La POO permite reutilizar codigo, representar situaciones reales(personas, productos) y matener el codigo organizado y facil de limpiar.

# EN RESUMEN
- La programacion lineal busca optimizar descisiones mediane modelos matematicos, mientras que la programacion orientada a objetos busca estructurar y organizar el codigo de forma logica y eficiente.
