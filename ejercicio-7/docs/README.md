# CineMax - Sistema de Cine (Semana 07)

## Dominio
Sistema de reservas para CineMax, cine premium en Bogotá.

## Excepciones Personalizadas
- `ReservaInvalidaException`: Datos inválidos (código, precio, cliente nulo)
- `FuncionNoDisponibleException`: Sin cupo o función pasada

## Características
- Estructura profesional por paquetes
- Validaciones con IllegalArgumentException y personalizadas
- 2 excepciones checked con constructores completos
- Main con 5 casos de prueba + finally
- Todo compila y ejecuta correctamente
