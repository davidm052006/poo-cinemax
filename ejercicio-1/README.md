### Descripción rápida
Sistema orientado a objetos para el cine CineMax que permite:
- Gestionar películas (título, precio, duración, si es estreno, etc.)
- Gestionar salas (número, tipo IMAX/3D, capacidad, Dolby, película asignada)
- Calcular recaudación estimada
- Asignar películas a salas
- Verificar disponibilidad de salas
