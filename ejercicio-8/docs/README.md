# CineMax - Semana 08: Colecciones y Programación Genérica

## Funcionalidades Nuevas
- Migración a `ArrayList` para historial de funciones
- `HashMap` para búsquedas rápidas por código
- `HashMap` bonus: Agrupación por película
- Filtrado por precio y por mes/año
- Estadísticas: Total ventas, promedio, máximo precio, conteo por película
- Menú interactivo con 6 opciones

## Requisitos Cumplidos
- No arrays de objetos (solo ArrayList)
- Generics en todas las colecciones
- CRUD completo
- Búsqueda O(1) con HashMap
- 2 métodos de filtrado
- Estadísticas completas
- Menú funcional
